# System Use Cases

## Actors
- Guest
- Host
- Admin
- Payment Gateway

## Key Interactions
1. **Guest**: Search properties ➔ Book ➔ Pay ➔ Review
2. **Host**: Create listing ➔ Manage bookings ➔ Respond to reviews
3. **Admin**: Moderate content ➔ View analytics
4. **System**: Send notifications ➔ Process payments
